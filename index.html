Добавить Cerberus

После добавления на главный экран будет видна иконка Cerberus и сайт откроется быстрее.

iPhone Safari

  1. Открой сайт в Safari.
  2. Нажми кнопку «Поделиться».
  3. Выбери «На экран Домой».
  4. Подтверди название Cerberus.

Android Chrome

  1. Открой сайт в Chrome.
  2. Нажми меню с тремя точками.
  3. Выбери «Добавить на главный экран».
  4. Подтверди установку ярлыка.

Windows

  1. Открой файл сайта в браузере.
  2. Создай ярлык на этот файл.
  3. Закрепи ярлык на рабочем столе.
